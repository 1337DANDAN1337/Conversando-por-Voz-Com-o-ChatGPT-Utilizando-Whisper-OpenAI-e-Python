# Conversando-por-Voz-Com-o-ChatGPT-Utilizando-Whisper-OpenAI-e-Python

Text to speech for streams

🎤💬 Chat por Voz Multi-Idiomas com Whisper + ChatGPT

📌 Sobre o Projeto

Este projeto foi desenvolvido no Google Colab com o objetivo de criar um sistema de conversa por voz utilizando Inteligência Artificial.

A solução integra tecnologias de:

Speech-to-Text (voz → texto)
Processamento de linguagem natural
Text-to-Speech (texto → voz)

Permitindo que o usuário fale com o sistema e receba respostas também em áudio, de forma inteligente e em múltiplos idiomas.

🚀 Como Funciona

O fluxo do sistema é simples e eficiente:

🎙️ O usuário faz uma pergunta por voz
🧠 O áudio é transcrito usando o modelo Whisper
💬 O texto é enviado para o ChatGPT
🤖 O ChatGPT gera uma resposta inteligente
🔊 A resposta é convertida em áudio com gTTS
🎧 O usuário escuta a resposta

🧠 Tecnologias Utilizadas

Google Colab
Whisper (OpenAI) – reconhecimento de fala (ASR)
ChatGPT (OpenAI) – geração de respostas inteligentes
gTTS (Google Text-to-Speech) – síntese de voz
Python

🌍 Diferenciais

Suporte a múltiplos idiomas
Alta precisão mesmo com sotaques e ruídos
Conversação totalmente por voz
Integração simples e executável em nuvem

▶️ Como Executar

Acesse o notebook no Google Colab
Execute todas as células
Grave ou envie um áudio
Ouça a resposta gerada automaticamente
